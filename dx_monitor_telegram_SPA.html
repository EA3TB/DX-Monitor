✈️ TELEGRAM BOT

Configuración para DX Monitor
Obtener Token · Chat ID · Imagen del bot
Esta guía explica cómo crear un bot de Telegram para recibir las alertas DX en tu móvil u ordenador. Si aún no tienes cuenta de Telegram, el primer paso explica cómo crearla.
1 Crear una cuenta de Telegram
ℹ Si ya tienes cuenta de Telegram, pasa directamente al Paso 2.

Descarga la app desde la tienda de tu dispositivo o desde la web oficial:

🌐 telegram.org

También puedes usar Telegram desde el navegador sin instalar nada:

🌐 web.telegram.org
  • Al registrarte solo necesitas un número de teléfono
  • Introduce el código que recibes por SMS
  • El nombre de usuario (que empieza por @) es opcional pero recomendable
2 Crear el bot con @BotFather

BotFather es el bot oficial de Telegram para gestionar bots. Es el único método oficial.

Abre Telegram y busca @BotFather en el buscador. Verifica que tiene la marca azul de verificación oficial. Pulsa Start o escribe /start.

Escribe /newbot y sigue las instrucciones:

Alright, a new bot. How are we going to call it?
Tu_indicativo DX Monitor
Good. Now let's choose a username for your bot. It must end in bot.
Tu_indicativo_DXMonitor_bot

El nombre puede ser cualquier cosa descriptiva. El username debe ser único en Telegram y terminar en bot.

BotFather te responderá con el token:

Done! Congratulations on your new bot. Use this token to access the HTTP API: 123456789:AABBcc...
⚠ Guarda el token en un lugar seguro. Es secreto: quien lo tenga controla tu bot.
3 Obtener tu Chat ID

El Chat ID identifica tu conversación personal con el bot. Es donde recibirás todas las alertas DX.

  • Busca tu bot por su username (@Tu_indicativo_DXMonitor_bot) y pulsa Start
  • Escribe cualquier mensaje al bot desde Telegram
  • Abre la siguiente URL en el navegador, sustituyendo el token:
https://api.telegram.org/bot<TU_TOKEN>/getUpdates

Verás un JSON con tu mensaje. El Chat ID está en:

"chat": { "id": 1045454849, ... }

Ese número es tu Chat ID personal. Cópialo.

4 Configurar en el Dashboard

En el dashboard de DX Monitor, sección Telegram:

  • Token bot → el token que te dio BotFather
  • Chat ID → el número obtenido en el paso anterior
  • Idioma alertas → Español o English

Pulsa Enviar mensaje de prueba para verificar que llega correctamente.

ℹ La URL del dashboard es http://<IP_DEL_HOST>:8765
5 Crear una imagen para el bot

BotFather permite asignar una foto de perfil al bot con el comando /setuserpic.

Recomendaciones de diseño:

  • Formato cuadrado, mínimo 640×640 px (Telegram lo redondea en círculo)
  • Fondo oscuro con el indicativo o texto DX Monitor
  • Icono de antena, ondas de radio o el símbolo de radioafición

Herramientas gratuitas:

  • Canva — plantillas fáciles, exporta PNG directamente
  • GIMP — editor de imagen local, gratuito
  • logomakr.com — opción rápida online

Para subir la imagen, escribe en BotFather:

/setuserpic
Choose a bot to change profile photo.
@Tu_indicativo_DXMonitor_bot
OK, send me a photo now.
[adjuntas la imagen]
Comandos útiles de BotFather
ComandoFunción
/mybotsVer y gestionar todos tus bots
/setnameCambiar el nombre visible del bot
/setdescriptionTexto que aparece al abrir el bot por primera vez
/setabouttextTexto breve en el perfil del bot
/setuserpicFoto de perfil
/deletebotEliminar el bot permanentemente
/revokeRevocar el token actual y generar uno nuevo
⚠ Usa /revoke inmediatamente si crees que tu token ha sido comprometido.
Resumen rápido
  1. Cuenta Telegram → telegram.org o app del móvil
  2. @BotFather → /newbot → guarda el TOKEN
  3. Escribe al bot → abre /getUpdates en el navegador → copia el CHAT_ID
  4. Dashboard DX Monitor → sección Telegram → pega token y chat_id → Test
  5. @BotFather → /setuserpic → sube tu imagen 640×640 px
